# User & Access Control - Quick Reference

## 📍 How to Access

**Navigation Path**: Admin Menu → Access Control

**Direct URL**: `/page/UserAccessControl`

## 🎯 Feature Access Tab

### What It Shows
- Complete feature availability matrix for all plans
- Which features each user has access to
- Visual breakdown of accessible vs restricted features

### How to Use
1. **View Matrix**: See all features and which plans have access
2. **Click on User Card**: Expand to see individual feature access
3. **Visual Indicators**:
   - ✓ Green checkmark = Feature available
   - 🔒 Lock icon = Feature restricted
4. **Status Badges**: Shows current plan and status (Active/Disabled)

### Features in Matrix
- Invoices, Quotes, Clients (Free+)
- Recurring Invoices (Starter+)
- Accounting, Reports (Professional+)
- Payroll (Enterprise only)
- Banking, Messages, Notes (All plans)

## 🔒 Account Suspension Tab

### Instant Suspension Features
- **Suspend Button**: Click to instantly disable account
- **Reactivate Button**: Click to restore access
- **Status Badge**: Shows current account status
- **Activity Log**: Track all suspension events

### What Happens When Suspended
✓ Cannot access any features or data
✓ Cannot log in to platform
✓ Sessions terminated immediately
✓ Can be reactivated anytime

### Activity Log
Shows:
- Who was suspended/reactivated
- When the action occurred
- Admin who performed action
- Status of change

## 🔄 Common Actions

### To Suspend a User Account
1. Go to Account Suspension tab
2. Find the user in the list
3. Click "Suspend" button (red outline)
4. Action takes effect immediately

### To Reactivate a User Account
1. Go to Account Suspension tab
2. Find the suspended user (appears with red highlight)
3. Click "Reactivate" button
4. User can log in again immediately

### To View Feature Access for User
1. Go to Feature Access tab
2. Click on any user card to expand
3. See all features with status (✓ or 🔒)
4. Green background = accessible, Gray background = restricted

### To Understand Feature Limitations
1. Go to Feature Access tab
2. Look at "Feature Availability by Plan" matrix
3. Find the feature in rows, read across columns
4. Check which plans have access (green checkmark)

## 🎯 Status Meanings

### User Status
- **Active** (green): User can access account and features
- **Disabled** (red): User cannot access anything

### Feature Status
- ✓ (green): Feature available for that plan
- 🔒 (gray): Feature not available for that plan

### Plan Levels
- **Free**: Basic features, 1 user limit
- **Starter**: Includes recurring invoices, 3 users
- **Professional**: Includes accounting & reports, 10 users
- **Enterprise**: All features, unlimited users

## ⚡ Key Points

✓ **Instant Effect**: All changes happen immediately
✓ **No Delays**: Suspensions take effect right away
✓ **Audit Trail**: Every action is logged
✓ **Admin Only**: Only admins can access this page
✓ **Visual Clear**: Color-coded badges for quick status
✓ **Reversible**: Suspended accounts can be reactivated

## 📋 Activity Log Information

The suspension activity log shows:
- **User Name**: Who was affected
- **User Email**: Contact information
- **Action**: Suspended or Reactivated
- **Timestamp**: Exact date and time
- **Performed By**: Admin who made the change
- **Badge**: Quick status indicator
